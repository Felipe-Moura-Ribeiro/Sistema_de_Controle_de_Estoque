# Sistema_de_Controle_de_Estoque

Este projeto, denominado "Sistema_de_Controle_de_Estoque", é uma aplicação Java. O sistema oferece funcionalidades básicas de controle de estoque, permitindo o cadastro de peças, filiais, realização de inserção, retirada, transferência e listagem de peças associadas às filiais.

## 1.Funcionalidades Principais
- Cadastro de Peça

  - Opção 1 no menu: Permite cadastrar uma nova peça no sistema, solicitando nome, tipo e quantidade.

- Cadastro de Filial

  - Opção 2 no menu: Permite cadastrar uma nova filial no sistema, associando-a à empresa.

- Inserção de Peças

  - Opção 3 no menu: Permite inserir peças em uma filial específica, solicitando o tipo de peça e a quantidade.

- Retirada de Peças

  - Opção 4 no menu: Permite retirar peças de uma filial específica, solicitando o tipo de peça e a quantidade.

- Transferência de Peças

  - Opção 5 no menu: Permite transferir peças entre filiais, solicitando a filial de origem, a filial de destino, o tipo de peça e a quantidade.

  - Listagem de Filiais e Peças

- Opção 6 no menu: Exibe uma listagem das filiais e suas peças associadas.

- Sair do Sistema

- Opção 7 no menu: Encerra a execução do sistema.

## 2.Como Executar
- Compile e execute o código Java ControledeEstoque.java no ambiente de desenvolvimento Java.
- Siga as opções do menu para interagir com o sistema de controle de estoque.
- Escolha a opção "7" para sair do sistema.

## 3.Observações
O projeto utiliza as classes Peca, Filial, e Empresa para representar peças, filiais e a empresa, respectivamente.
A interação com o usuário é realizada por meio da classe Scanner.
O código possui verificações de tipo de peça duplicada e outras funcionalidades que podem ser adaptadas conforme necessário.
Este projeto foi desenvolvido como parte de um exercício para implementar um sistema de controle de estoque simples em Java. Sinta-se à vontade para utilizar, modificar e expandir conforme as necessidades do seu próprio projeto.
